# Beam Studio 功能地圖

> FLUX 雷射機器的隨附軟體。資料來源：測試表（331 項）、程式碼、語言檔 `en.ts`／`zh-tw.ts`。測試覆蓋詳情見 `docs/tests/<分類>/Summary.md`。英文版：`feature-map.en.md`。
> 入口與元件層級（功能住在哪一層／哪個元件）見 → `docs/ui-entry-map.zh-TW.md`。

## 平台與後端

| | Web（PWA） | 桌面版（Electron） |
|---|---|---|
| 多分頁、原生選單、最近使用檔案、自動儲存 | — | ✓ |
| Swiftray 引擎（Promark、路徑引擎選項） | —（寫死關閉） | ✓ |
| 其餘全部功能 | ✓ | ✓ |

後端服務：**FLUXGhost**（WebSocket API——SVG/PDF/AI 解析、工作/gcode 產生、相機、機器探索；隨桌面版啟動時為動態埠）、**Swiftray**（6611 埠——Promark 與替代轉檔引擎）、**FLUX ID 雲端**（`id.flux3dp.com`——帳號、My Cloud、AI 點數、Google Fonts 代理、S3 字型檔）。

## 支援機型

beamo (fbm1)、beamo II (fbm2)、Beambox／Pro (fbb1b/p)、Beambox II (fbb2)、HEXA／HEXA RF (fhexa1)、Ador (ado1，模組化：10W/20W 二極體、2W 紅外、列印頭)、Lazervida (flv1)、Promark (fpm1)、Miro UV (fuv1)

## 1. 檔案操作
- **匯入**：JPG、PNG、SVG†、DXF、PDF†、AI†、.beam、.bvg；拖曳匯入；SVG 匯入分層選擇視窗（依圖層／依顏色／不分層；選項組合依模組而異）
- **匯出／儲存**：.beam、.bvg、SVG、PNG、JPG；儲存／另存新檔；最近使用檔案（桌面版）；自動儲存（桌面版；間隔與檔案數上限）
- **範例**：各機型內建範例專案與材質測試檔

## 2. 畫布與繪圖
- 圖形（矩形含圓角、橢圓、多邊形±邊數、直線）、鋼筆／貝茲曲線與路徑編輯模式
- **元素庫 Elements**：內建圖庫＋ **Noun Project** 線上圖庫整合（分類／搜尋／釘選；登入後解鎖免費圖示——文案為「登入取得 10,000+ 免費圖示」、新功能公告標示「超過一百萬個圖形」，非「700 萬」）
- 文字：系統字型＋內建網頁字型＋ **Google Fonts 面板**（搜尋／語言／類別）；字級、字距、行距、直書；路徑文字；**變數文字**（流水號/CSV）
- 選取（單選／多選／Shift 角度鎖定）、變形（移動/旋轉/縮放、鎖定比例）、群組／解散、複製／貼上／原地貼上、復原／重做
- **產生器**：QR Code（容錯率、反轉）與條碼、**Boxgen 盒子產生器**（指接/T 槽/平接、上蓋、內外尺寸）、**材質測試格**（速度×功率掃掠）、鑰匙圈、拼圖、印章

## 3. 物件操作
- 尺寸／座標／旋轉輸入；布林運算（合併/相減/相交/差異）；**位移複製**（向內外、轉角樣式；點陣圖以外框矩形處理）；**橋接 tabs**（防掉件，含手動模式與間距）；陣列；鏡像；解散非連續路徑；**合併文字**；文字轉路徑（逐字選項、字型替換警告）
- 影像操作：漸層開關、曲線、銳化、裁剪、生成斜角、色彩反轉、向量化、取代影像；**圖像編輯視窗**（橡皮擦／魔術棒／圓角，Konva）；**印章產生器 Stamp Maker**（影像轉印章：斜角半徑、水平翻轉、濾鏡復原／重做）
- **智慧排版 Auto Fit**（自動排到相機辨識的工件上）、自動對齊線與吸附、**深度模式**（漸層點陣圖 → 最小功率選項）

## 4. 圖層
- 新增/刪除/複製、逐圖層參數（功率/速度/次數…）、顏色、拖曳排序、合併（向下/全部）、鎖定、顯示/隱藏、跨圖層搬移與參數提示
- **Ador 列印圖層**：全彩／單色、CMYK 展開、雷射↔列印模組切換提示視窗、預噴區；白墨 White Ink（dev 模式限定）；UV 列印（dev 限定／Miro UV）

## 5. 參數與預設
- 各機型/模組材質預設；新增/匯入/匯出/重置自訂參數
- 防呆：向量速度上限（20 mm/s；BB2 為 50）紅色滑桿警告、低功率提示（<10%）、送工作時高功率確認（beamo/BB2）、列印進階模式

## 6. 視圖
- 縮放（按鈕/滾輪/觸控板）、平移（中鍵拖曳/空白鍵拖曳）、配合視窗、隨視窗縮放、格線、尺標、圖層顏色檢視、反鋸齒
- 各機型工作範圍畫布；Ador 各模組邊界疊層；開蓋（open-bottom）模式

## 7. 相機
- 預覽（拖曳範圍）、影像描圖、背景透明度 0–100%（五段）、清除預覽、匯入後保留；各機型校正精靈；自動對焦高度提示；相機資料備份

## 8. 工作產生與路徑預覽
- 轉檔引擎：FLUXGhost/beamify（web 全部工作）· Swiftray（桌面版 Promark／可選路徑引擎）
- **路徑預覽**：播放與倍速、行進路徑/反轉顯示、從這裡開始（算圖加速開啟時隱藏，桌面版）、尺寸/時間估算（與畫布右下角一致）、切割順序先內後外
- 影響工作的文件設定：**雕刻 DPI**（125/250/500/1000）、**自動內縮**（HEXA、≥250 DPI、1 像素侵蝕）、旋轉軸模式與倍率、工作原點、混合雷射偏移（預設右 70 / 上 7 mm）、自動對焦、Z 軸速度測試
- **Pass-through 大幅面分段雕刻**（BB2／Ador）：輸入物件長度 → 自動將工作範圍依高度切段；**參考圖層**（預設功率 0，不雕刻，僅供對齊）；**導引標記 Guide Mark**（雕刻於畫布作為對段對齊基準，可設長度與 X 座標）；輸出至工作範圍

## 9. 機器整合
- **初始連線設定精靈**（`pages/InitializeMachine`）：選擇機型 → 選擇連線方式（Wi-Fi／有線乙太／直連 Wired／USB，視機型而定）→ 依步驟設定 Wi-Fi、輸入機器 IP（含連線提示與下一步驗證）；Promark 專屬流程（選雷射源、Promark 設定、相機校正初始化）
- 機器探索（網路掃描）、機器 IP 工具與網路檢測
- **儀表板 Monitor**：工作進度、剩餘時間、相機、機上檔案瀏覽；開始/暫停/停止/結束
- **外框預覽**：外框（可見圖層範圍）／凸包／區域檢查；Ador 弱出光（藍光）
- 旋轉軸（chuck/roller、鏡像、擴展工作範圍）、自動送料、門蓋保護、韌體更新、連線測試、Promark 設定（雷射源、紅點、Z 軸、振鏡校正）

## 10. 3D 曲面雕刻（BB2）
- 框選範圍 → 對焦點網格 → 探測（參考 Z＋逐點自動對焦）→ 網格預覽含 >45° 紅點 → 點紅點看失敗原因 → 選點重新測量 → 曲面補償雕刻

## 11. 帳號與雲端
- FLUX ID：email＋Google/Facebook OAuth 登入、記住登入、AI 點數顯示
- **My Cloud**：儲存/另存/重新命名/複製/刪除/排序、縮圖、免費方案 5 檔上限
- AI 一鍵去背（點數計費，見 §13）；Design Market（dmkt.io）；社群連結；公告面板與評分提示

## 12. 偏好設定、新手引導與其他
- 23 種介面語言；單位 mm/inch；預設字型/機型/文件設定；參考線；自動儲存設定；更新檢查；Sentry 錯誤回報；重置所有設定
- **歡迎頁 Welcome**：最近使用檔案、My Cloud、Help Center、Follow Us 分頁；促銷橫幅（依地區）、範例／新建檔案捷徑、教學引導卡
- **新手引導**：初始流程（語言 → 機型選擇 → 連線設定，見 §9）；**新手教學 New User Tutorial**（引導畫布操作）＋**介面介紹 Interface Tutorial**（相機預覽、選擇機器、開始工作、基本圖形、鋼筆、新增圖層等逐步導覽）；影片教學（Media Tutorial）；說明選單（文件、論壇、更新紀錄、快捷鍵）

## 13. AI 與智慧功能
- **BEAMY AI 小助手**（`components/Chat`）：入口位於左側工具列（DrawingToolButtonGroup → ai-chat 抽屜）與歡迎頁；內嵌 udify.app 聊天機器人 iframe，依當前機型載入對應機型的問答助手（beamo／Beambox／BB2／HEXA／Promark／Ador…），提供操作與支援問答（標註「Beamy 可能會出錯，請查核重要資訊」）
- **AI 產圖 AiGenerate**（`components/AiGenerate`）：文字轉圖（提示詞、風格選擇、尺寸選擇、多圖數量、雷射友善 Laser-friendly 開關、生成歷史）；以 **AI 點數** 計費（每張固定成本，點數不足時彈出購買提示，連結至 FLUX ID 購買頁）
- **AI 一鍵去背**（背景移除，點數計費）
> AI 功能需登入 FLUX ID 並消耗雲端 AI 點數。
