# 參數與預設 (presets-parameters)

## 本分類測試範圍
涵蓋雷射與列印參數預設的操作，以及功率、速度等警告提示，包含：切換參數改變下方功率選項、新增與匯入匯出參數、管理與恢復預設值、自訂參數清單、功率過高與功率過低的警告提示、Ador 彩色圖層的濃度速度層數五段程度設定與手動輸入，以及白底（White Ink）切換。UV 墨水限速與雙面列印屬 web 版不存在的功能，降低焦距與遞降需自動對焦機型實機，皆維持人工。

## 測試檔案清單

| 測試檔案 | 類型 | 涵蓋內容摘要 | 來源 |
| --- | --- | --- | --- |

| `laser-panel.spec.ts` | Cypress E2E | 切換參數改變功率選項、新增與匯入匯出參數、管理與恢復預設、自訂參數清單 | 既有 |
| `Ador-layer.spec.ts` | Cypress E2E | Ador 彩色圖層進階模式開關、濃度速度層數五段程度顯示與手動輸入 | 既有 |

## 尚未自動化項目

| 項目 | 原因 |
| --- | --- |
| UV Ink 限速（30mm/s、10mm/s） | UV 列印為 dev-gate 的獨立模組或另一台機器（Miro UV），並非 Ador 上的墨水類型選項，web 版無對應的速度上限 UI 可斷言。 |
| 雙面列印 | web 程式碼庫中不存在此功能（無 twoSided/two_sided/雙面 原始碼），無法自動化。 |
| 功率過高警告（BB2、Beamo） | 警告只在連線機器並送出工作時觸發，屬機器行為，UI 層無法自動化。 |
| 降低焦距於「自動對焦」機型 | 需自動對焦機型實機，維持人工，納入發版前實機檢查清單。 |
| 遞降功能於「自動對焦」機型 | 需自動對焦機型實機，維持人工，納入發版前實機檢查清單。 |
