# 送工作 (job-execution)

## 本分類測試範圍

涵蓋送出工作後的執行控制與監看（測試表中「送工作」以兩個群組出現，此處合併）：停止功能、暫停與繼續功能、預估時間與實際工作時間是否接近，以及 Dashboard 上送出工作後的進度條與剩餘時間顯示是否正確。

本分類項目多屬機器行為或需連上實機，尚無專屬的自動化測試。

## 測試檔案清單

目前無自動化測試 — 本分類項目多屬機器行為或需連上實機才能驗證，尚無專屬 spec 覆蓋。

## 尚未自動化項目

- 停止功能是否正常 — 屬機器行為，維持實機人工。
- 暫停與繼續功能是否正常 — 屬機器行為，維持實機人工。
- 確認預估時間與實際工作時間是否接近 — 預估時間的計算邏輯可以 Jest 單元測試；與實際工作時間的比對需連上實機（Tier B／C）。
- 確認 Dashboard：送出工作後進度條與剩餘時間顯示是否正確 — 需連上實機的半自動化本地批次（Tier B）。
