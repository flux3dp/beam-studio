# 外框預覽 (frame-preview)

## 本分類測試範圍

涵蓋外框預覽功能：隱藏的圖層不進行外框預覽、外框預覽的三種模式（外框／凸包／區域檢查）、外框預覽的開始與停止以及關閉視窗是否正確，以及 Ador 外框弱出光（藍光）。

本分類項目多屬機器行為，需以實機驗證，尚無專屬的自動化測試。

## 測試檔案清單

目前無自動化測試 — 本分類項目多屬機器行為，需以實機驗證，尚無專屬 spec 覆蓋。

## 尚未自動化項目

- 隱藏圖層不納入外框預覽 — 屬計算邏輯，可自動化（Tier A／A3），建議以 Jest 或本地 FLUXGhost 環境的 Cypress 驗證；實際動作維持實機人工。目前尚未覆蓋。
- 外框預覽的三種模式（外框／凸包／區域檢查） — 屬機器行為，維持實機人工，納入發版前實機檢查清單。
- 外框預覽的開始與停止是否正確、關閉視窗是否正確 — 屬機器行為，維持實機人工。
- Ador 外框弱出光（藍光） — 屬機器行為，維持實機人工。
