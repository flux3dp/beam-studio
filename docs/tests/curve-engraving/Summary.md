# 3D 曲面雕刻功能 (curve-engraving)

## 本分類測試範圍

涵蓋 BB2 的 3D 曲面雕刻流程：框選範圍 → 選擇對焦數量並輸入物件高度 → 取得參考點 → 開始對焦 → 對焦後送出工作。重點檢查項目包含對焦完的 3D 網狀格顯示是否正確、超過 45 度處是否顯示紅點、取得參考點失敗與對焦失敗的錯誤處理、重新測量，以及送工作前的負重測試與 Z 軸防撞。

其中屬純數學與純狀態流轉的部分（>45° 紅點判斷、`maxAngle` 計算、對焦資料的狀態旗標）以 Jest 涵蓋；實際的對焦流程、3D 網狀格顯示與機器行為則維持元件測試（A3）或實機人工。

## 尚未自動化項目

- 整體對焦流程（框選 → 對焦數量 → 輸入高度）與取得參考點失敗、對焦失敗紅點點擊顯示原因、點擊對焦點重新測量、完成後再次預覽 3D 曲面 — 屬 UI 流程，建議以 Jest 元件測試 `CurveEngraving/MeasureArea` 涵蓋（A3）；實際流程維持實機人工。
- 送出 3D 工作前的負重測試、負重測試後移到雕刻位置前先於原點下降 Z 軸防撞、完成 3D 工作時再次下降 Z 軸防撞 — 皆屬機器行為，維持實機人工（Tier C）。
- 雕刻時間與實際工作時間是否相近 — 時間比對需實機；預估邏輯可併入送工作分類的預估時間單元測試。
